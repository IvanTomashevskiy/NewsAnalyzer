Об авторе

Здравствуйте. Меня зовут Иван, я - Фронтенд-разработчик и автор News Analyzer.

Данное приложение позволяет:

❶ Осуществлять поиск новостей по ключевым словам за последние семь дней.

❷ Автоматически сохранять и отображать, при последующем посещении, результаты поиска.

❸ Строить гистограмму аналитики по упоминанию ключевых слов в новостях и заголовках за последние семь дней.

Применяемый стек технологий: JS, ES6, БЭМ, ООП, CSS, HTML, NPM, Webpack, GIT. Также, применялись следующие библиотеки: Fetch API, GitHub API, News API, Swiper API.
